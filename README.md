# Autonomous AI Automation Pipeline

An orchestration architecture utilizing n8n, PostgreSQL, and pgvector for AI workflows and RAG (Retrieval-Augmented Generation) capabilities. 

## Architecture Stack
* **Orchestration:** n8n (Self-hosted)
* **Database:** PostgreSQL
* **Vector Store:** pgvector (pg16)
* **Execution Environment:** Python 3.x

## Prerequisites
* Docker and Docker Compose
* Python 3.10+
